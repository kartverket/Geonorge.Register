# Geonorge.Register

Registeret er et verktøy for å administrere registre/kodelister. 

## Kom i gang

### Forutsetninger

- .net framework 4.8

### Oppsett

1. Klon repoet:
   ```bash
   git clone git@github.com:kartverket/Geonorge.Register.git
   cd Geonorge.Register
   ```

2. Installer avhengigheter:
   ```bash
   yarn install
   ```
